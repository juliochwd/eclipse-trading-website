## 📝 System Components

### Eclipse Trading System

```
├── eclipse_scanner.py      # Market scanner (15 min interval)
├── monitor.py              # Real-time trade monitoring
├── heartbeat_sender.py     # System health checker
├── gold_watchdog.sh        # Auto-heal script
├── eclipse_config.py       # Configuration
└── data/
    ├── trades.json           # Trade history
    └── stats.json            # Performance stats
```

### Systemd Services
- `eclipse-scanner.timer` - Scan every 15 minutes
- `eclipse-monitor.service` - 24/7 monitoring
- `eclipse-heartbeat.timer` - Health check every 3 hours
- `eclipse-watchdog.timer` - System watchdog every 5 minutes
